Get an annual summary of the performance of storm overflows in England. This data is provided by Water and Sewerage Companies (WaSCs) and published by the Environment Agency. There are datasets for each year since 2020 that show the:

- number of storm overflows with event duration monitors (EDM)
- number of storm overflows returning spill counts
- average spill duration and average spill count for each WaSC 

There’s also a dataset with long-term trends. You can download each dataset as a ZIP file or view the data on a map, with the ability to filter by year and WaSC.
