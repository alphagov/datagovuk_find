# Support
The Government Digital Service (GDS) maintains the service behind data.gov.uk. This includes:

- the redesigned data.gov.uk pages that were published in March 2026, including the curated data collections and data manual
- the previous version of data.gov.uk which we’ve renamed the data.gov.uk directory, and which includes functionality to find and publish data 

The data.gov.uk service cannot:

- provide personal information or medical records
- combine data from multiple datasets
- host data 

## If you’re a civil servant 
You can join the conversation about data.gov.uk on our UK Government Digital Slack channel: [#datagovuk](https://ukgovernmentdigital.slack.com/archives/C037J3GTE4T).

## If you have a question about the new features 
If you have a question about the curated data collections, the data manual or our roadmap, please complete our [feedback form](https://forms.office.com/e/9V26PNFQaR). 

## If you want to propose new data for the data collections 
Tell us about it on our [feedback form](https://forms.office.com/e/9V26PNFQaR). 

## If you have a question about a dataset on the data directory 
Contact the publisher directly using the contact details on the dataset’s page.  

## Other requests  
If your request is urgent or you’d like to report an issue, please use the [GOV.UK support form](https://www.gov.uk/contact/govuk) and mention that it’s about data.gov.uk. Use this link if you need help with publishing on the directory too.

For questions about GOV.UK, check the [GOV.UK help pages](https://www.gov.uk/help) or [contact GOV.UK](https://www.gov.uk/contact/govuk).

The NHS [publishes its own data](https://digital.nhs.uk/data-and-information/data-collections-and-data-sets/data-sets).
