Check the past results of a vehicle’s MOT tests, including if it passed or failed and the mileage recorded when it was tested. Search for a vehicle by entering its registration number. You can also check when the next MOT is due and download a copy of the current and previous MOT certificates. For tests in England, Scotland or Wales, you’ll be able to see where each test was done, and what parts failed or had minor problems. For tests in Northern Ireland, you can only get results for tests done since 2017. 

The MOT history API combines all vehicle types into a single API based on REST principles. The response data is in JSON format and uses standard HTTP error response codes.
