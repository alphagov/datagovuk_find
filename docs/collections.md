# Collection and dataset ordering

This document lists the collections in alphabetical order, as they appear on home
page and navigation.

Within a collection the datasets are listed in priority order.

The list of datasets is based on the intial curated dataset document and subsequent content document.

It's possible that not all the datasets/topics listed below will be in final mvp, and not all items are present yet in `app/content` but will be soon.

If we drop an item from the list, then it can be removed from this doc.
